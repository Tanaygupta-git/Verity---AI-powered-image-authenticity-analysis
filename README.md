Verity is a modern web-based application designed to analyze and detect whether an image is AI-generated or real.

The system analysis techniques to evaluate visual patterns, textures, and inconsistencies, providing users with an authenticity score on a scale of 0 to 10. Along with the score, the application presents a clear label (e.g., Likely Real or Possibly Generated) and concise explanations to improve transparency.

Key Features:
- Upload and analyze images in real-time
- Authenticity scoring system (0–10 scale)
- Clear result classification with confidence level
- Explainable output with key detection reasons
- Clean, minimal, and premium user interface
- Fast and responsive performance

Tech Stack:
- Frontend: Modern UI (React / HTML / Tailwind CSS)
- Backend: API-based image analysis
- Logic: Feature-based detection (texture, lighting, patterns, Meta data, Exif data)

This project focuses on simplicity, usability, and explainability, making AI-powered image detection accessible to everyday users.
